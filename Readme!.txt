SISTEM REKOMENDASI FILM - HBF
==============================

📌 DESKRIPSI SINGKAT
---------------------
Aplikasi ini adalah sistem rekomendasi film menggunakan metode Hybrid-Based Filtering (HBF). Aplikasi dibuat dengan Python dan Streamlit sebagai antarmuka pengguna.

🔗 LINK CEPAT
-------------
▶ Akses langsung aplikasi (online):
   https://sistemrekomendasifilm-hbf-set-based.streamlit.app/

📁 Repository GitHub:
   https://github.com/alfiiinur/SistemRekomendasiFilm-HBF.git

=================================================
🔧 CARA MENJALANKAN APLIKASI SECARA LOKAL (OFFLINE)
=================================================

1. 💻 **PERSYARATAN SISTEM**
   - Python 3.8 atau lebih baru
   - Pip (Python package manager)
   - Koneksi internet (untuk install dependensi)

2. 📥 **CLONE REPOSITORY**
   Buka terminal (CMD, PowerShell, atau terminal Linux/macOS), lalu jalankan:
    --> git clone https://github.com/alfiiinur/SistemRekomendasiFilm-HBF.git
    --> cd SistemRekomendasiFilm-HBF

3. 📦 **BUAT & AKTIFKAN VIRTUAL ENVIRONMENT (OPSIONAL tapi disarankan)**
- Windows:
  ```
  python -m venv env
  env\Scripts\activate
  ```
- macOS/Linux:
  ```
  python3 -m venv env
  source env/bin/activate
  ```

4. 📚 **INSTALL DEPENDENSI**
Jalankan:
    --> pip install -r requirements.txt


5. 🚀 **JALANKAN APLIKASI**
    --> streamlit run app.py


6. 🌐 **AKSES DI BROWSER**
Setelah dijalankan, Streamlit akan membuka aplikasi di browser secara otomatis.
Jika tidak, buka manual:
    --> http://localhost:8501


7. 🛑 **UNTUK KELUAR**
Tekan `CTRL + C` di terminal.
